ICON FOLDER - Place Your Custom Icons Here
===========================================

This folder is for your custom icon images to replace the emoji placeholders throughout the website.

REQUIRED ICON FILES:
-------------------

Logo Icons:
- clinic-logo.png (32x32px or 64x64px)

Feature Icons (Home Page):
- expert-dentists.png (128x128px)
- easy-booking.png (128x128px)
- appointment-management.png (128x128px)
- comprehensive-care.png (128x128px)

Service Icons:
- orthodontics.png (48x48px or 64x64px)
- cosmetic-dentistry.png (48x48px or 64x64px)
- restorative-dentistry.png (48x48px or 64x64px)
- oral-surgery.png (48x48px or 64x64px)
- dentures.png (48x48px or 64x64px)
- dental-xrays.png (48x48px or 64x64px)

FORMATS:
--------
- PNG (recommended - transparent background)
- SVG (best for scalability)
- WebP (modern, good compression)
- JPG (if no transparency needed)

HOW IT WORKS:
-------------
1. Add your icon files to this folder with the exact names above
2. The website will automatically load your icons
3. If an icon file is missing, the emoji placeholder will show instead

For more details, see ICON_GUIDE.md in the project root folder.
